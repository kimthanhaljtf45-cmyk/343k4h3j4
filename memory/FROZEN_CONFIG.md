## ROLES (ЗАМОРОЖЕНО)

Публічний UX:
- "Для дитини" → PARENT
- "Для себе" → STUDENT

Внутрішні ролі:
- PARENT - батьки учнів
- STUDENT - дорослі учні (самооборона)
- COACH - тренери
- ADMIN - адміністратори

## SUBSCRIPTION STATES (ЗАМОРОЖЕНО)

- ACTIVE - активна підписка
- PAUSED - пауза (відпустка, хвороба)
- CANCELLED - скасована

---

## PAYMENT STATES (ЗАМОРОЖЕНО)

- PENDING - очікує оплати
- UNDER_REVIEW - на перевірці (чек завантажено)
- PAID - оплачено
- REJECTED - відхилено
- OVERDUE - прострочено (автоматично)

---

## CONSULTATION FUNNEL (ЗАМОРОЖЕНО)

1. NEW - новий лід
2. CONTACTED - зв'язались
3. BOOKED_TRIAL - записаний на пробне
4. TRIAL_DONE - пробне пройшло
5. CONVERTED - конвертований в учня
6. LOST - втрачений

---

## DASHBOARD PHILOSOPHY (ЗАМОРОЖЕНО)

- Mobile = ACTION-FIRST (щоденні дії)
- Web Admin = BUSINESS-FIRST (управління)
- Alerts = SYSTEM-FIRST (автоматика)

---

## TECH STACK (ЗАМОРОЖЕНО)

- Backend: NestJS + MongoDB
- Mobile: Expo / React Native
- Web Admin: Next.js (планується)
- Intelligence: MetaBrain (NestJS)
